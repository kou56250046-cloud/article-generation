# chapter-split — 章分割 Skill

長い原典テキストを、物語化に適した章単位に分割・整理する。

## 使い方

```
/chapter-split
[長文テキストをここに貼り付け]
```

---

## 出力フォーマット

```
## 章分割案

全体テーマ：[原典全体が伝えようとしていること]
推奨分割数：[X話]

---

### 第1話（推奨）
- **タイトル案**: 「[タイトル]」
- **対応原典箇所**: [原典の該当部分]
- **中心テーマ**: [この話で扱う主要概念]
- **弟子の問い**: [この話の導入となる疑問]
- **分量目安**: [短め / 普通 / 長め]

### 第2話
...
```

---

## 分割の基準

1. **概念の完結性**: 一つの問いと答えが完結する単位で区切る
2. **難易度の順序**: 易しい概念から難しい概念へ段階的に並べる
3. **一話の長さ**: 対話往復4〜6回に収まる量を目安とする
4. **接続性**: 前の話の気づきが次の話の問いにつながるよう設計する
